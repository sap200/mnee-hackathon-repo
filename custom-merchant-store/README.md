# Rapyd MNEE Website Integration Example

This website is live on [http://rapydmnee.duckdns.org:9000/](http://rapydmnee.duckdns.org:9000/)

This repository contains example source code showing how to integrate the **Rapyd MNEE Payment Gateway** into your own website.

It demonstrates:

- Creating checkout sessions
- Connecting with the Rapyd MNEE backend services

## Prerequisites

Before using this example, make sure you have:

- **Rapyd MNEE Dashboard** running ([https://github.com/rapyd-mnee-payment-gateway/mnee-stripe](https://github.com/rapyd-mnee-payment-gateway/mnee-stripe))
- **Payment Service** running ([https://github.com/rapyd-mnee-payment-gateway/paymentservice.git](https://github.com/rapyd-mnee-payment-gateway/paymentservice.git))
- **MNEE Service** running ([https://github.com/rapyd-mnee-payment-gateway/mnee-service](https://github.com/rapyd-mnee-payment-gateway/mnee-service))

These services handle API calls, sessions, and transactions.

## Usage

1. Explore the source code in this repository to see how the integration works.
2. Use it as a reference for your own website implementation.
3. For full API reference and advanced usage, check the **Docs section** inside the **Rapyd MNEE Dashboard**.

> This example is meant to help you learn and implement Rapyd MNEE payments quickly on your website.
